# Tasty Crousti Runner 🍱

## Structure du projet
```
tasty-crousti-runner/
├── index.html          ← le jeu complet
└── images/
    ├── logo.png        ← logo Tasty Crousti (fond transparent)
    ├── player.jpg      ← la barquette (personnage du joueur)
    ├── rice.jpg        ← bol de riz (obstacle)
    ├── sauce.jpg       ← pot de sauce (obstacle)
    └── onions.png      ← oignons frits (fond noir) (obstacle)
```

## Comment ajouter vos images
1. Sauvegardez chaque image avec le bon nom dans le dossier `images/`
2. Ouvrez `index.html` dans un navigateur (double-clic)
3. Les images s'affichent automatiquement → sinon des formes colorées s'affichent à la place

## Contrôles
- **WASD** ou **Flèches** : déplacer la barquette (haut/bas/gauche/droite)
- **ESPACE** : démarrer / recommencer

## Fonctionnalités
- Vue de dessus (top-down)
- 3 vies avec images de cœurs
- Multiplicateur de score (combo jusqu'à ×5)
- Invincibilité après un coup (effet clignotant)
- Particules (effet d'éclaboussure de nourriture)
- Screen shake à chaque coup
- Vitesse qui augmente progressivement
- Obstacles en zigzag + obstacles rapides
- Meilleur score mémorisé
